# Ecoboom Website Scraping Summary

## Site Structure
The website is a multi-page site built on Squarespace. It consists of the following main pages:
- **Home**: Main landing page with an overview of services and mission.
- **About**: Detailed "what we do" section.
- **Services**: Overview of specific services (Sound & Light, DJ & Rentals, Mobile Solar).
- **Mission**: Detail on environmental goals and impact.
- **Testimonials**: Client feedback.
- **Contact**: Inquiry form and location details.
- **Rentals / Events / Mobile Solar Power**: These appear to link back to sections or sub-pages within the main structure.
